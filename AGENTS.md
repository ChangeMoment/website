# Home Page
On the homepage, the first hero section should include a large full-screen image that covers the entire first viewport. The image should feel soft and atmospheric.

The hero image itself has a fade effect to the left edges.

When the homepage loads, animate the hero image with a very slow, subtle zoom-in effect over a few seconds. The animation should feel calm and elegant, not distracting.

# General
Use a very light background color across the website to create a calm, clean, and welcoming feeling.

Use a dark footer with light-colored text for strong contrast and a professional finish.

The content of the hero section should be placed on the left side of the screen, not too close to the left border of the screen. 
This will ensure that the text is easy to read and stands out against the softer background.

The non-hero sections use very light gradient backgrounds that blend smoothly from one section to the next, instead of abrupt white/pale-blue bands.
